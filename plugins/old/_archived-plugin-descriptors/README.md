# Archived Runtime Plugin Marker Directories

This folder stores historical `.codex-plugin/` and `.claude-plugin/` metadata directories moved out of their original archived-package positions so recursive plugin/marketplace scanners do not treat `plugins/old/` as active installable runtime surface.

These descriptors are provenance only. Do not install from here unless intentionally resurrecting a legacy package in a separate migration pass.

Current active replacement for the CPF/Ariadne lineage is:

```text
../../Aeriadne/
```
