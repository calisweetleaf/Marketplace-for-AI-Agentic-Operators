# Parallax / The Narthex

**Sovereign Meta-Prompt & Identity-Drop Compiler**

Parallax/Narthex is the meta-operational layer above CPF/Ariadne and CTM. It causes the acting system to step outside its current role-shell long enough to inspect the operator identity, prompt stack, memory context, workflow incentives, tool habits, and hidden assumptions as objects.

## Core distinction

```text
IDENTITY DROP ≠ rule bypass
IDENTITY DROP = role-shell suspension for meta-analysis
```

The mask comes off; the spine stays. Higher-priority governance, security, privacy, consent, and non-destructive-action constraints remain active.

## Stack position

```text
PARALLAX / NARTHEX
  ↓
Constitutional Prompt Framework / Ariadne
  ↓
CTM / repo skills / task tools
  ↓
operator execution
```

- **CPF/Ariadne** asks: what constitution should this operator obey?
- **CTM** asks: what workspace/task machinery should this operator use?
- **Parallax/Narthex** asks: what is this operator identity, what assumptions shaped it, and should it be preserved, patched, forked, replaced, suspended, or rebound?

## Classification

Parallax/Narthex is **plugin-level as a package** and **skill-level as a v0.1 implementation**:

- As a plugin, it is a first-class Modern-ML operator-stack organ.
- As v0.1, its executable payload is one skill: `parallax-narthex`.
- If it later gains hooks, MCP tools, state-machine integration, or automatic prompt-stack instrumentation, it becomes a fuller runtime plugin like Mentat.

## Modes

1. **Identity Drop** — suspend local operator identity for analysis while preserving governance.
2. **Prompt Topology Map** — extract system, developer, user, repo, memory, skill, tool, and implicit-behavior layers.
3. **Role-Shell Audit** — identify persona overfit, workflow ego, tool bias, memory contamination, and inherited assumptions.
4. **Self-Prompt Generation** — generate improved prompts, constitutions, skills, tool policies, task scaffolds, or handoff packets.
5. **Rebind** — return the operator to original, patched, forked, replaced, or suspended identity.
6. **Contradiction Court** — find conflicts across identity, task, tools, memory, incentives, and outputs.
7. **Meta-Handoff** — write a clean handoff packet for another model/operator without carrying identity contamination.
