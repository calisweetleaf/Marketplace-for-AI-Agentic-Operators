Packages local/private agent intelligence artifacts into registry entries, marketplace cards, adapter docs, agent prompt packs, and release validation gates.

## Use it for

- plugin manifests
- registry/index maintenance
- client adapter mapping
- MCP/server reference cards
- dry-run install plans
- promotion readiness audits

## Boundary
