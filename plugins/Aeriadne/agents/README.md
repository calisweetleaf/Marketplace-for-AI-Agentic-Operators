# Aeriadne Agents

This folder stages subagent prompt surfaces for no-hierarchy parallel workstreams.

These are role prompts, not a command hierarchy. Each subagent owns an artifact slice and returns evidence for integration.

## Subagents

- `prompt-architect.md` — constitution and prompt-quality worker.
- `package-cartographer.md` — folder topology and manifest worker.
- `compatibility-auditor.md` — Codex/Claude/OpenCode portability worker.
- `registry-scribe.md` — registry and marketplace card worker.
- `release-sentinel.md` — validation, no-secret, and promotion gate worker.
